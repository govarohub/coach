# Coach

Aplicación móvil desarrollada en Flutter para conectar usuarios con entrenadores personales mediante geolocalización, reservaciones y pagos electrónicos.

## Tecnologías

- Flutter
- Firebase
- Cloud Firestore
- Firebase Storage
- Google Maps
- Mercado Pago
- Riverpod
- GoRouter
